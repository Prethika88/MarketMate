## **MarketMate**

MarketMate is a full-stack online grocery shopping platform designed to deliver a seamless and secure digital shopping experience. It allows users to easily browse a wide range of grocery products, add and manage items in their shopping carts, and complete purchases through a reliable and secure online payment system. The platform uses modern web technologies to ensure smooth performance, intuitive navigation, and secure transactions for both users and administrators.


## **Tech Stack**
- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** Google OAuth  
- **Payments:** Stripe API


  
## **Features**

- User-friendly grocery shopping experience with product browsing and cart management. 
- Secure checkout with 100% Stripe payment integration and webhook support. 
- Dynamic Admin Dashboard for real-time product and order management.  
- Scalable backend architecture for smooth operations.


## **Admin Capabilities**
- Add, update, and delete products. 
- Monitor and manage customer orders.
- Real-time data synchronization with the database.


## **Security & Payments**
- Google OAuth for secure authentication.
- Stripe Webhooks for reliable payment confirmation and order processing.
